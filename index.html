Paramètres Google API

Saisissez votre Client ID Google OAuth. C'est nécessaire pour autoriser l'application à accéder à Google Drive.

Nécessaire si vous hébergez vous-même l'application sur un autre domaine.

API Gemini (IA)

Saisissez votre clé d'API Gemini pour activer l'assistant d'import automatique par photo ou lien.

Votre clé est stockée uniquement en local sur votre appareil.

Serveur Proxy Google Apps Script

Optionnel. Saisissez l'URL de votre Web App Google Apps Script pour contourner les protections CORS et améliorer la capture des images et des recettes.

Si non renseigné, l'application utilisera des proxies publics et l'IA locale.

Requis uniquement si vous avez configuré la propriété ACCESS_TOKEN dans Google Apps Script.
